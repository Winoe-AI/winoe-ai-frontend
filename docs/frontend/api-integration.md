# API Integration Matrix (Canonical)

Source of truth:

- Frontend: `src/features/**`, `src/app/api/**`, `src/platform/**`
- Backend cross-reference: `../winoe-backend/app/**/routes/**`

Status legend:

- `verified`: frontend path/method maps to implemented backend route.
- `mismatch`: integration exists but request/response contract does not fully align with backend implementation.
- `frontend-only`: no backend route mapping is expected (local-only utility/debug route or external URL upload).

## Notes

- Backend API prefix is configured as `/api` in `../winoe-backend/app/config/config_settings_fields_config.py`.
- Frontend `/api/backend/[...path]` requests are translated to backend `/api/{path}` in `src/platform/server/backendProxy/target.ts`; `/api/backend/*` client paths are therefore expected for proxy-backed calls.
- Trial routes are mounted under `/api/trials/*` in backend route registry `../winoe-backend/app/trials/routes/trials_routes/__init__.py`.
